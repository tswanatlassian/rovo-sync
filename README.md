# Rovo Sync

**Automated Confluence ↔ Jira Sync System**

Keeps your Confluence planning pages in sync with Jira work items automatically.

## Features

- 🔄 **Hourly Sync** - GitHub Actions runs every hour
- 💡 **Decision Capture** - Automatically extracts decisions from comments
- 📊 **Change Detection** - Monitors status changes, blockers, completions
- 📚 **Learning Loop** - Builds organizational memory over time
- 🚀 **Fully Automated** - Set it and forget it

## Quick Start

See [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md) for complete setup instructions.

## How It Works

1. **Planning** - Create outcome-driven plans in Confluence
2. **Execution** - Team works in Jira
3. **Sync** - System automatically syncs every hour
4. **Learning** - Captures decisions and learnings for future

## Built With

- Python 3.11
- GitHub Actions
- Atlassian APIs (Jira + Confluence)
